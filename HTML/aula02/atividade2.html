HTML
É a linguagem padrão para a criação de páginas web. Ela define a estrutura e o conteúdo de um site, utilizando elementos (tags) para organizar texto, imagens, vídeos e outros tipos de conteúdo
CSS
Linguagem de estilo utilizada para definir a apresentação visual de documentos HTML, como páginas web
Bootstrap
É um framework front-end de código aberto e gratuito, usado para desenvolver websites e aplicações web responsivas, com foco em design mobile-first
JavaScript
Linguagem de programação de script utilizada para adicionar interatividade e elementos dinâmicos a páginas web e aplicações
React
É uma biblioteca JavaScript de código aberto, focada no desenvolvimento de interfaces de usuário (UIs) interativas para aplicações web, móveis e outras. Ele permite a construção de UIs complexas a partir de componentes reutilizáveis e gerenciados por si próprios, promovendo a organização e eficiência no desenvolvimento.
