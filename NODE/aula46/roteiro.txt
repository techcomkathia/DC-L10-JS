Passo a passo para criação da API

- Garantir que o Banco de Dados existe
- Configurar o projeto no diretório:
    - inicialização do node 
    - instalação das dependências : 
        - nodemon
        - express
        - sequelize
        - mysql2
    - criação das pastas:
        - modelo
        - serviços
        - controladores
        - rotas
        - middlewares
    - configurar o acesso ao banco e os modelos
        - configBanco
        - usuariosModel
        - produtosModel
        - servidorBanco
    - testar a conexão com o Banco e a sincronização das tabelas
    - criar aquivo main e configurar um servidor para o express
    - Ordem das camadas (mais profunda até a mais superficial. Mais profunda que é mais próxima ao banco de dados e a mais superficial que é mais próxima das requisições do usuário) modelo/serviços/controladores/rota/servidorAPI
    
    - configurar os serviços
        -testar individualmente os serviços em um arquivo a parte
    - configurar os controladores
    - configurar os middlewares de validação
    - configurar as rotas
    - usar as rotas no arquivo principal do servidor da aplicação (main.js)

    -AULA 45 
        -instalar o bcrypt 
        -criação de senhas seguras:
            - na camada de serviço, criar o hash de senha antes de passar para o método create do modelo
        - verificação de senhas para rotas protegidas:
            - comparar a senha enviada no corpo da requisição com o hash de senha armazenado no banco de dados

    -IMPORTANTE: para testar as rotas com qualquer método diferente do GET usar o postman 

    -AULA 46
        - Instalar o dotenv
        - Criar o arquivo .env
        - Identificar as informações que devem estar seguras (e os arquivos que elas são utilizadas)
        e criar as variáveis com os valores necessários
        - Utilizar as variáveis de ambiente substituindo as informações presentes nos arquivos necessários
